WordPress plugin to prevent extra image sizes from being generated. 

== Description ==

This WordPress Plugin prevents extra, and usually unnecessary, images sizes from being generated. 

* No setup required!

== Installation ==

Upload the "wp-prevent-generating-extra-image-sizes" folder to your wp-content/plugins directory, then Activate it.
